# Habilidades Generales
- **Resolución de Problemas:** Capacidad analítica para depurar código y optimizar procesos.
- **Trabajo en Equipo:** Experiencia coordinando proyectos integradores académicos.
- **Comunicación Asertiva:** Habilidad para documentar procesos técnicos de forma clara.
- **Pensamiento Crítico:** Evaluación de diferentes arquitecturas o soluciones antes de la implementación.
- **Colaboración en Ecosistemas Ágiles:** Experiencia trabajando bajo metodologías que priorizan la entrega de valor, el feedback constante y la cohesión del equipo.

- **Atención al Detalle y Calidad:** Enfoque riguroso en la depuración de código y en la validación de interfaces para asegurar una experiencia de usuario sin fricciones.

- **Empatía con el Usuario Final:** Capacidad para ponerse en el lugar del usuario, fundamental para diseñar soluciones accesibles e inclusivas.

- **Gestión del Tiempo y Priorización:** Habilidad para organizar cargas de trabajo académicas y proyectos de desarrollo, cumpliendo con los deadlines establecidos.
